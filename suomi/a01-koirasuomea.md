# Koirasuomea 1

## Keskustelu

- Saako moikata?
- Onks se uros vai narttu?
- Tää on tyttö.
- Okei, no sitten joo. Tää ei oikeen tykkää toisista uroksista.

## Sanasto

| Suomeksi | Selitys | 
| ----------- | ----------- | 
| moikata | to say ’hi’ | 
| uros (urokse-) | male | 
| narttu | bitch | 
| tää | tämä | 
| ei oikein (ei oikeen) |  not really | 
| toinen (toise-) |  other |
